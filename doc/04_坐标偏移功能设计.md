# 坐标偏移功能设计文档

## 1 功能概述

### 1.1 需求背景

在自动化场景中，经常遇到以下情况：
- OCR识别到文字位置，但实际需要点击的是文字下方的按钮
- 图像匹配到图标位置，但需要操作的是图标旁边的输入框
- 颜色检测到某个区域，但需要记录的是该区域内的特定点

因此需要在条件节点检测到位置后，通过偏移量记录实际要操作的位置。

### 1.2 设计目标

1. **简单易用**：用户无需学习复杂表达式，直接配置偏移值
2. **可视化配置**：提供测量工具帮助用户快速获取偏移量
3. **灵活模式**：支持相对偏移和绝对坐标两种模式
4. **向后兼容**：不影响现有行为树文件，默认偏移为0

## 3 详细设计

### 3.1 ConditionNode 基类扩展

在 `bt_core/nodes.py` 的 `ConditionNode` 类中添加以下属性和方法：

```python
class ConditionNode(Node):
    def __init__(self, node_id=None, config=None):
        super().__init__(node_id, config)
        # ... 现有属性 ...
        
        # 新增偏移参数
        self.offset_x = self.config.get_int("offset_x", 0)
        self.offset_y = self.config.get_int("offset_y", 0)
        self.offset_mode = self.config.get("offset_mode", "relative")
    
    def _apply_offset(self, position: tuple) -> tuple:
        """应用坐标偏移
        
        Args:
            position: 原始位置 (x, y)
            
        Returns:
            tuple: 偏移后的位置
        """
        if position is None:
            return None
        
        if self.offset_mode == "relative":
            return (position[0] + self.offset_x, position[1] + self.offset_y)
        else:  # absolute
            return (self.offset_x, self.offset_y)
    
    def _save_position(self, context, position: tuple):
        """保存位置到黑板（应用偏移）
        
        Args:
            context: 执行上下文
            position: 原始位置
        """
        if position and self.save_position:
            final_position = self._apply_offset(position)
            context.blackboard.set(self.position_key, final_position)
```

### 3.2 条件节点修改

各条件节点需要将原来的：
```python
context.blackboard.set(self.position_key, position)
```

改为：
```python
self._save_position(context, position)
```

## 4 偏移模式说明

### 4.1 相对偏移模式 (relative)

```
原始位置: (100, 200)
偏移量: X=+50, Y=+30
最终位置: (150, 230)

适用场景:
- OCR识别文字后点击下方按钮
- 图像匹配后点击相邻元素
- 颜色检测后操作附近控件
```

### 4.2 绝对坐标模式 (absolute)

```
原始位置: (任意)
偏移量: X=500, Y=300
最终位置: (500, 300)

适用场景:
- 检测到某个标志后，点击固定位置
- 忽略检测结果，使用预设坐标
```

## 6 序列化格式

### 6.1 JSON 格式示例

```json
{
  "id": "node_ocr_1",
  "type": "OCRConditionNode",
  "name": "OCR检测-开始按钮",
  "config": {
    "region": [0, 0, 800, 600],
    "keywords": "开始游戏",
    "language": "chi_sim",
    "offset_x": 0,
    "offset_y": 50,
    "offset_mode": "relative",
    "position_key": "last_detection_position"
  }
}
```

### 6.2 向后兼容

- 旧版本行为树文件不包含偏移参数
- 反序列化时使用默认值 (offset_x=0, offset_y=0, offset_mode="relative")
- 不影响现有行为树的执行
