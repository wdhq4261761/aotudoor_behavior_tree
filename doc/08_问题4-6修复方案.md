## 问题4：重复代码

### 4.1 问题分析

**GUI 模块中的重复代码**：

1. **节点删除逻辑重复**：`_delete_selected` 和 `_delete_nodes` 方法存在重复的保护节点检查逻辑
2. **复制粘贴逻辑重复**：单节点和多节点的复制粘贴处理存在大量重复代码

**位置定位**：
- [bt_gui/bt_editor/editor.py:340-406](bt_gui/bt_editor/editor.py#L340-L406) - 删除节点方法
- [bt_gui/bt_editor/editor.py:408-500](bt_gui/bt_editor/editor.py#L408-L500) - 复制粘贴方法

**问题影响**：
- 代码维护困难
- 修改时容易遗漏
- 增加代码体积

### 4.2 修复策略

1. **提取公共方法**：将重复的逻辑提取为私有方法
2. **统一处理入口**：使用统一的入口方法处理单节点和多节点情况
3. **使用模板方法模式**：定义操作模板，具体实现由子类完成

### 4.3 修复实现

**文件**: `bt_gui/bt_editor/editor.py`

```python
def _check_protected_nodes(self, node_ids: List[str]) -> Tuple[List[str], List[str]]:
    """检查节点是否受保护
    
    Args:
        node_ids: 要检查的节点ID列表
    
    Returns:
        (可删除节点列表, 受保护节点列表)
    """
    deletable = []
    protected = []
    
    for node_id in node_ids:
        node_item = self.canvas.nodes.get(node_id)
        if node_item and node_item.is_protected():
            protected.append(node_id)
        else:
            deletable.append(node_id)
    
    return deletable, protected


def _show_protected_warning(self, protected_count: int) -> None:
    """显示受保护节点警告
    
    Args:
        protected_count: 受保护节点数量
    """
    if protected_count > 0:
        messagebox.showwarning("无法删除", "开始节点不可删除")


def _delete_selected(self):
    """删除选中的节点或连接"""
    if self.canvas.selected_nodes:
        node_ids = list(self.canvas.selected_nodes)
        self._delete_nodes_with_check(node_ids)
    elif self.canvas.selected_node:
        self._delete_nodes_with_check([self.canvas.selected_node])
    elif self.canvas.selected_connection:
        self._delete_connection(self.canvas.selected_connection)


def _delete_nodes_with_check(self, node_ids: List[str]) -> None:
    """删除节点（带保护检查）
    
    Args:
        node_ids: 要删除的节点ID列表
    """
    deletable, protected = self._check_protected_nodes(node_ids)
    self._show_protected_warning(len(protected))
    
    if not deletable:
        return
    
    if len(deletable) == 1:
        command = RemoveNodeCommand(canvas=self.canvas, node_id=deletable[0])
    else:
        command = RemoveNodesCommand(canvas=self.canvas, node_ids=deletable)
    
    self.command_manager.execute(command)
    self._update_toolbar()
    self._set_modified(True)


def _delete_connection(self, connection: Tuple[str, str]) -> None:
    """删除连接
    
    Args:
        connection: (parent_id, child_id) 元组
    """
    parent_id, child_id = connection
    command = RemoveConnectionCommand(
        canvas=self.canvas,
        parent_id=parent_id,
        child_id=child_id
    )
    self.command_manager.execute(command)
    self.canvas.selected_connection = None
    self._update_toolbar()
    self._set_modified(True)
```

## 问题5：资源泄漏风险

### 5.1 问题分析

**ScriptExecutor 类的资源管理问题**：

1. **缺少上下文管理器**：没有实现 `__enter__` 和 `__exit__` 方法
2. **缺少析构方法**：没有实现 `__del__` 方法作为最后保障
3. **用户可能忘记调用 shutdown**：导致线程池资源泄漏

**位置定位**：
- [bt_utils/script_executor.py:258-263](bt_utils/script_executor.py#L258-L263) - shutdown 方法

**问题影响**：
- 线程池资源泄漏
- 长时间运行可能导致资源耗尽
- 异常情况下无法正确清理

### 5.2 修复策略

1. **实现上下文管理器协议**：支持 `with` 语句
2. **添加析构方法**：作为资源清理的最后保障
3. **使用弱引用回调**：确保资源被正确释放

### 5.3 修复实现

**文件**: `bt_utils/script_executor.py`

```python
import weakref
import atexit


class ScriptExecutor:
    """脚本执行器
    
    支持上下文管理器协议，确保资源正确释放。
    
    使用方式:
        # 方式1：上下文管理器（推荐）
        with ScriptExecutor() as executor:
            executor.run_script(script)
        
        # 方式2：手动管理
        executor = ScriptExecutor()
        try:
            executor.run_script(script)
        finally:
            executor.shutdown()
    """
    
    _instances = weakref.WeakSet()
    
    def __init__(self):
        self._script_thread: Optional[threading.Thread] = None
        self._stop_event = threading.Event()
        self._pause_event = threading.Event()
        self._executor_pool = ThreadPoolExecutor(max_workers=1)
        self.is_running = False
        self.is_paused = False
        self._script_content: Optional[str] = None
        self._loop = False
        self._lock = threading.Lock()
        
        ScriptExecutor._instances.add(self)
    
    def __enter__(self) -> "ScriptExecutor":
        """进入上下文管理器"""
        return self
    
    def __exit__(self, exc_type, exc_val, exc_tb) -> bool:
        """退出上下文管理器，确保资源释放"""
        self.shutdown()
        return False
    
    def __del__(self):
        """析构方法，作为资源清理的最后保障"""
        try:
            self.shutdown()
        except Exception:
            pass
    
    @classmethod
    def cleanup_all(cls) -> None:
        """清理所有实例（用于程序退出时）"""
        for instance in list(cls._instances):
            try:
                instance.shutdown()
            except Exception:
                pass


# 注册退出清理函数
atexit.register(ScriptExecutor.cleanup_all)
```

## 问题6：配置管理冗余

### 6.1 问题分析

**配置类的冗余设计**：

1. **BlackboardConfig 和 SessionConfig**：只是简单的数据容器
2. **功能与 SettingsManager 重叠**：SettingsManager 已经支持嵌套字典
3. **增加维护成本**：需要同时维护多个类

**位置定位**：
- [config/settings_manager.py:12-36](config/settings_manager.py#L12-L36) - BlackboardConfig
- [config/settings_manager.py:40-61](config/settings_manager.py#L40-L61) - SessionConfig

**问题影响**：
- 代码冗余
- 维护成本增加
- 可能导致数据不一致

### 6.2 修复策略

由于这两个配置类已经在项目中被广泛使用，直接删除可能导致兼容性问题。建议采用以下策略：

1. **保留配置类**：作为类型安全的访问层
2. **简化实现**：移除冗余的转换方法
3. **添加类型注解**：提高代码可读性
4. **添加缓存机制**：避免重复创建配置对象

### 6.3 修复实现

**文件**: `config/settings_manager.py`

```python
from typing import Any, Dict, Optional, List, Callable
from dataclasses import dataclass, field, asdict
import json
import os
import datetime
from bt_utils.singleton import singleton


VERSION = "1.0.0"


@dataclass(frozen=True)
class BlackboardConfig:
    """黑板配置（不可变）"""
    default_position_key: str = "last_detection_position"
    default_value_key: str = "last_number_value"
    default_ocr_text_key: str = "last_ocr_text"
    default_color_key: str = "last_color_value"
    default_image_match_key: str = "last_image_match"


@dataclass(frozen=True)
class SessionConfig:
    """会话配置（不可变）"""
    last_file_path: str = ""
    recent_files: tuple = field(default_factory=tuple)
    window_geometry: str = "1280x800"
    last_export_path: str = ""
    
    def __post_init__(self):
        # 确保 recent_files 是元组
        if isinstance(self.recent_files, list):
            object.__setattr__(self, 'recent_files', tuple(self.recent_files[:10]))


@singleton
class SettingsManager:
    """统一配置管理器
    
    使用单例模式，线程安全。
    支持类型安全的配置访问。
    """
    
    VERSION = "1.0.0"
    CONFIG_FILE_NAME = "config.json"
    
    # 配置缓存
    _blackboard_config_cache: Optional[BlackboardConfig] = None
    _session_config_cache: Optional[SessionConfig] = None
    
    # ... 其他代码保持不变 ...
    
    def get_blackboard_config(self) -> BlackboardConfig:
        """获取黑板配置（带缓存）"""
        if self._blackboard_config_cache is not None:
            return self._blackboard_config_cache
        
        data = self.get("blackboard", {})
        self._blackboard_config_cache = BlackboardConfig(
            default_position_key=data.get("default_position_key", "last_detection_position"),
            default_value_key=data.get("default_value_key", "last_number_value"),
            default_ocr_text_key=data.get("default_ocr_text_key", "last_ocr_text"),
            default_color_key=data.get("default_color_key", "last_color_value"),
            default_image_match_key=data.get("default_image_match_key", "last_image_match"),
        )
        return self._blackboard_config_cache
    
    def set_blackboard_config(self, config: BlackboardConfig) -> None:
        """设置黑板配置"""
        self._blackboard_config_cache = config
        self.set("blackboard", asdict(config))
    
    def get_session_config(self) -> SessionConfig:
        """获取会话配置（带缓存）"""
        if self._session_config_cache is not None:
            return self._session_config_cache
        
        data = self.get("session", {})
        self._session_config_cache = SessionConfig(
            last_file_path=data.get("last_file_path", ""),
            recent_files=tuple(data.get("recent_files", [])[:10]),
            window_geometry=data.get("window_geometry", "1280x800"),
            last_export_path=data.get("last_export_path", ""),
        )
        return self._session_config_cache
    
    def set_session_config(self, config: SessionConfig) -> None:
        """设置会话配置"""
        self._session_config_cache = config
        self.set("session", asdict(config))
    
    def invalidate_config_cache(self) -> None:
        """使配置缓存失效"""
        self._blackboard_config_cache = None
        self._session_config_cache = None
```
